**Online Assessment System (Django)**

A comprehensive web-based platform designed to streamline the creation, delivery, and analysis of multiple-choice assessments. This project features a robust administrative dashboard for content management and a user-friendly interface for candidates.

**Key Features**

**Admin Module**

Assessment Management: Create, update, and delete assessments with ease.

Question Bank: Add and manage MCQ questions linked to specific assessments.

User Management: Full CRUD (Create, Read, Update, Delete) functionality for candidate profiles.

Advanced Reporting: * View raw assessment data and individual candidate performance.

Export to PDF: Generate and download professional result reports.

**Candidate Module:**

Secure Testing: Dedicated portal for candidates to take assigned assessments.

Intuitive UI: Clean interface for navigating through MCQ questions.

Instant Submission: Automated scoring and data logging upon completion.


HOW TO USE
1) create a folder named "onlineassessment"
2) clone the project to your system
3) create mysql database 
4) Add your database name, username, password in database fields in base or development settings
5) create a super user
6) Try login with the super user credential
